# Projet Dichotomie

Bienvenue sur le dépôt du projet **Dichotomie**. Ce projet implémente l'algorithme de dichotomie en Python, accompagné de sa documentation théorique/énoncé au format PDF.

## Description

L'objectif de ce projet est de démontrer le fonctionnement de la méthode de dichotomie (ou *bisection method*). Cet algorithme est couramment utilisé pour :
* Trouver des valeurs approchées de solutions d'équations (ex: f(x) = 0).
* Effectuer des recherches efficaces dans des listes triées (complexité logarithmique).

Le dépôt contient à la fois le code source de l'implémentation et le fichier PDF détaillant le contexte ou l'énoncé du problème.

## Structure du dépôt

* `*.py` : Le script Python contenant l'implémentation de l'algorithme.
* `*.pdf` : Document de référence (énoncé, rapport ou explications mathématiques).
* `.gitignore` : Fichier de configuration pour exclure les fichiers inutiles du versionning.

## Prérequis

Pour exécuter le code de ce projet, vous avez besoin de :
* **Python 3.x** installé sur votre machine.

## Installation et Exécution

1.  **Cloner le dépôt**
    ```bash
    git clone https://github.com/YassineT-git/Dichotomie.git
    cd Dichotomie
    ```

2.  **Lancer le script**
    Exécutez la commande suivante dans votre terminal :
    ```bash
    python demo_numerique.py
    ```
    *(Remplacez `demo_numerique.py` par le vrai nom du fichier Python présent dans le dossier)*
